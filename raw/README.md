# raw/ — Source documents

Drop your source documents here: articles, papers, notes, transcripts, images, data files.

**These files are immutable.** The LLM reads from them but never modifies them — they are
your source of truth. To add knowledge to the wiki, drop a source here and ask the LLM to
*ingest* it.

On ingest, the LLM will evaluate the source's credibility (see `METHODOLOGY.md` → CRAAP),
disclose any bias, and write it back into `wiki/` with cited, confidence-tagged claims.

Tip: the Obsidian Web Clipper browser extension converts web articles to markdown for quick
capture into this folder.
